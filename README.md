# Usage
```
npm start -- <arguments>

Possible arguments:

    -p, --price: Price in Yen to find under (default 1500)
    -n, --pages: Number of pages (90 skeb users per page) to search through (default 5)
    -a, --art: Include to search for artists
    -v, --voice: Include to search for voice actors
    -t, --text: Include to search for writers
    --no-nsfw: Include to ignore nsfw skebbers
```
